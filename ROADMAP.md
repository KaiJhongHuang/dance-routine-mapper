# Roadmap

## #1 印刷模式

匯出圖表為圖片，支援 Web Share API（手機可直接分享到 ibon / LINE），桌機則下載 PNG。

- [x] html2canvas 截圖 + Web Share API
- [ ] 進階：html2canvas + jsPDF 匯出 PDF

Issue: [#1](https://github.com/KaiJhongHuang/dance-routine-mapper/issues/1)

## #2 拖拉排序

在圖表或動作順序區直接拖拉動作來調整順序，拖拉結束後自動同步更新文字輸入。

Issue: [#2](https://github.com/KaiJhongHuang/dance-routine-mapper/issues/2)

## #3 快捷範本

提供常見步法組合（6-count basic、8-count swing out 等）一鍵插入，範本包含預設子步法。

Issue: [#3](https://github.com/KaiJhongHuang/dance-routine-mapper/issues/3)

## #4 分享庫

使用者可發布舞序到公開庫，其他人可瀏覽、搜尋、一鍵載入。

Issue: [#4](https://github.com/KaiJhongHuang/dance-routine-mapper/issues/4)

## #5 多語言

支援英文介面，讓國際 swing dance 社群也能使用。介面文字抽離為語言檔，支援語言切換。

Issue: [#5](https://github.com/KaiJhongHuang/dance-routine-mapper/issues/5)

## #6 回饋機制

在頁面上提供回饋入口，讓使用者能方便地提交建議或回報問題。

Issue: [#6](https://github.com/KaiJhongHuang/dance-routine-mapper/issues/6)
