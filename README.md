# Dance Routine Mapper

排舞圖表產生器 — 在 Google 試算表撰寫舞序，一鍵同步產生色彩對照表格。

**線上使用：** https://kaijhonghuang.github.io/dance-routine-mapper/

## 功能

- **Google 試算表整合（v2.0 主軸）**：貼上 sheet 連結，工作表分頁 = 一支舞，點分頁切換、一鍵同步
- 同色代表同一個動作，箭頭 `↓` 表示動作跨越 8 拍邊界延續
- 用 `T` 標記段落標題，支援多段落舞步
- **子步法記譜**：A 欄空、B 欄拍數、C 欄動作名 → 自動變成縮排子步法
- **「顯示細節」切換**：一鍵切換總覽 / 細節檢視模式
- **即時編輯（進階）**：折疊區內仍保留文字編輯、拖拉排序、快捷範本等原有功能
- **匯出圖片**：截取圖表為 PNG，手機支援原生分享
- **分享連結**：將舞序壓縮為 URL，一鍵分享

## 輸入格式

第一行為舞序名稱，`T` + `標題` 分隔段落，每行 `拍數` + `動作名稱`：

```
Lindy Hop 基本組合
T 第一段
6 6 Count Basic
6 Send Out
8 Break

T 第二段
8 Swing Out
8 Texas Tommy
```

### 子步法

在動作下方用縮排標記每段拍的步法細節：

```
6 Send Out
  2 Rock Step
  2 Triple Step
  2 Triple Step
```

點擊「顯示細節」按鈕後，圖表會展開為雙列顯示：
- 上方：主動作名稱（完整跨格）
- 下方：子步法（拆分小格、較小字體、較淺底色）

沒有子步法的動作不受影響，完全向下相容。

## Google 試算表整合

將舞序存在 Google Sheet、在 App 一鍵載入（單向：Sheet → App）。每個工作表分頁 = 一支舞。

### 快速開始

1. 下載範本：[雙欄+子步法](templates/lindy-hop-template.csv) 或 [單欄簡易](templates/lindy-hop-simple.csv)
2. Google Sheets：檔案 → 匯入 → 上傳 → 選剛下載的 CSV → 匯入資料
3. 檔案 → 共用 → **發布到網路** → 發布（只公開檢視，不影響編輯權限）
4. 複製 sheet 連結，在頁面最上方「Google 試算表整合」面板貼上，按「儲存連結」
5. 按「自動偵測分頁」抓出所有工作表分頁（只支援發布後的連結）
6. 點分頁鈕切換不同舞序，按「從試算表載入」同步

### 多舞序（分頁）

- 每個工作表（Sheet1, Sheet2, ...）代表一支舞
- 發布後按「自動偵測分頁」會列出所有分頁
- 也可以手動按「+ 新增分頁」輸入 `name + gid`
- gid 在網址的 `gid=XXX` 參數，第一個分頁通常是 `0`

### Sheet 格式

每列一行舞序文字，支援三種填法：

| A 欄 | B 欄 | C 欄 | 對應輸入 |
|---|---|---|---|
| `6 Send Out` | | | `6 Send Out` |
| `6` | `Send Out` | | `6 Send Out` |
| `T` | `第一段` | | `T 第一段` |
| （空） | `2` | `Rock Step` | `  2 Rock Step`（子步法） |

連結只存在瀏覽器的 localStorage（`drm:sheet-url`），不會外傳。

## 版本紀錄

### v2.0

- **使用方向轉變**：從「在網頁撰寫」改為「在 Google 試算表撰寫，網頁同步產生圖表」
- Google 試算表整合拉到頁面最上方，預設展開
- 原即時編輯區改為第二順位折疊區塊（「即時編輯（進階）」）
- 試算表分頁切換：工作表分頁 = 不同舞序，一鍵切換
- 「自動偵測分頁」：從發布的 pubhtml 抓出所有工作表名稱與 gid
- 預設 sheet 連結（首次打開無 localStorage 時）

### v1.2

- 頁面版面重新排列：圖表 → 動作順序 → 文字輸入區
- 舞序名稱（輸入第一行），顯示於圖表上方
- 匯出圖片：html2canvas + Web Share API，手機可直接分享
- 分享連結：deflate 壓縮 + URL-safe Base64，短網址分享
- 拖拉排序：動作順序區可拖拉，支援跨段落搬移，手機觸控支援
- 快捷範本：拍數佔位符（2/4/6/8 拍）與常見動作（含子步法）一鍵插入
- 手機編輯輔助：換行、刪行按鈕
- 複製按鈕移至輸入區下方

### v1.1

- 新增子步法記譜語法（縮排子行）
- 新增「顯示細節」切換按鈕
- 圖表細節模式：主動作在上、子步法在下，雙列顯示
- 動作順序區同步顯示子步法
- 預設範例加入子步法示範

### v1.0

- 圖表產生器：輸入動作序列，即時產生 8 拍色彩對照表格
- 支援多段落（`T` 標題分隔）與跨拍動作顯示（`↓` 箭頭）
- 內建經典排舞預設：Shim Sham（4 段）、Big Apple（2 段）
- Big Apple 動作順序已校正，依據 Frankie Manning 編舞的 Keep Punching (1939) 版本
